Expand description

Push config items to the given market config buffer account.

This instruction allows the authority to add new configuration items to their market config buffer account. The buffer will be reallocated to accommodate the new items, with the authority paying for any additional rent.

  • The authority must have enough SOL to pay for any additional rent needed.
  • The keys in new_configs must be valid MarketConfigKey.
