# GMSOL
## Development
### Prerequisites
- Rust
- Solana
- Anchor
- Node.js
- `pnpm`

### Installation
Install dependencies:
```bash
pnpm install
```

### Commands
To run all tests:
```bash
anchor test
```

To install the `gmsol` CLI:
```bash
pnpm install-cli
```
Use the following command to verify it is installed properly:
```bash
gmsol --version
```
