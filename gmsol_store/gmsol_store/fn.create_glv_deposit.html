  • The owner must be a signer and have sufficient balance for the execution fee and rent.
  • The store must be properly initialized.
  • The market must be:
    • Properly initialized
    • Owned by the store
    • Listed in the glv
  • The glv must be:
    • Properly initialized
    • Owned by the store
  • The glv_deposit must be:
    • Uninitialized
    • Address must be PDA derived from the SEED of GlvDeposit, store, owner and nonce
  • The glv_token must be:
    • Properly initialized
    • Correspond to the provided glv
  • The market_token must be:
    • Properly initialized
    • Correspond to the provided market
  • Token mint account requirements:
    • initial_long_token must be provided if initial long amount > 0
    • initial_short_token must be provided if initial short amount > 0
  • Escrow account requirements:
    • glv_token_escrow must be:
      • Owned by the glv_deposit
    • Other escrow accounts must be:
      • Provided for any non-zero initial token amounts
      • Owned by the glv_deposit
  • Source token account requirements:
    • Must be provided for any non-zero initial token amounts
    • Must have sufficient balance
    • Must have the owner as its authority
  • All token programs must match their corresponding token accounts
